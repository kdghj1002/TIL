# C언어
- 서로다른 CPU 명령체계에서 같은 소스의 프로그램을 구동시키기 위해서 개발된 언어.
- '데니스 리치'가 개발.
- 각각의 CPU 명령체계에 맞는 컴파일러로 C언어를 기계어로 변환함.


# C언어의 장점
1. 이식성이 높은 언어.
    - CPU의 종류에 상관없이 실행가능.
2. 절차지향적 언어
    - '정해진 순서의 실행흐름'을 중시하는 언어
3. 좋은 성능
    - 사용하는 메모리가 적어, 속도가 빠름


# C언어의 단점
- LOW-LEVEL의 특징을 함께 가지고 있어, HIGH-LEVEL언어보다 주의해서 사용.
